📋 音乐源

GET /api/sources

获取所有已注册的音乐源列表及默认音乐源。

curl http://localhost:8866/api/sources

响应示例：

{
  "success": true,
  "data": {
    "sources": ["QQMusicClient", "NeteaseMusicClient", ...],
    "default_sources": ["MiguMusicClient", "NeteaseMusicClient", ...],
    "total": 43
  }
}

🔍 搜索音乐（同步）

POST /api/search

同步搜索音乐，阻塞等待所有音乐源返回结果。

参数类型必填说明
keywordstring必填搜索关键词
music_sourcesstring[]可选指定音乐源列表，如 ["NeteaseMusicClient"]
init_music_clients_cfgobject可选客户端初始化配置（work_dir 等）
clients_threadingsobject可选各客户端搜索线程数
requests_overridesobject可选HTTP 请求参数覆盖（headers、proxies 等）
search_rulesobject可选搜索规则
curl -X POST http://localhost:8866/api/search \
  -H "Content-Type: application/json" \
  -d '{"keyword": "周杰伦 晴天", "music_sources": ["NeteaseMusicClient"]}'

🔍 搜索音乐（异步）

POST /api/search/async

异步搜索音乐，立即返回 task_id，通过 /api/tasks/{task_id} 查询结果。参数同同步搜索接口。

curl -X POST http://localhost:8866/api/search/async \
  -H "Content-Type: application/json" \
  -d '{"keyword": "周杰伦 晴天"}'

# 查询结果
curl http://localhost:8866/api/tasks/{task_id}

⬇️ 下载音乐

POST /api/download

下载指定歌曲（异步任务）。song_infos 从搜索结果中获取。

参数类型必填说明
song_infosobject[]必填SongInfo 字典列表（从搜索结果的 song_info 字段获取）
music_sourcesstring[]可选指定音乐源
init_music_clients_cfgobject可选客户端配置
clients_threadingsobject可选下载线程数
requests_overridesobject可选请求参数覆盖
curl -X POST http://localhost:8866/api/download \
  -H "Content-Type: application/json" \
  -d '{"song_infos": [从搜索结果中获取的 song_info]}'

🎯 搜索并下载

POST /api/search_and_download

一键搜索并自动下载（异步任务）。自动筛选有效下载链接的歌曲。

参数类型必填说明
keywordstring必填搜索关键词
max_downloadint可选最多下载数量（默认 5）
music_sourcesstring[]可选指定音乐源
curl -X POST http://localhost:8866/api/search_and_download \
  -H "Content-Type: application/json" \
  -d '{"keyword": "周杰伦 晴天", "max_download": 3}'

📜 解析歌单

POST /api/playlist/parse

解析歌单 URL，返回歌曲列表（同步）。

参数类型必填说明
playlist_urlstring必填歌单链接
music_sourcesstring[]可选指定音乐源
curl -X POST http://localhost:8866/api/playlist/parse \
  -H "Content-Type: application/json" \
  -d '{"playlist_url": "https://music.163.com/#/playlist?id=7583298906"}'

📜 歌单下载

POST /api/playlist/download

解析歌单并下载所有歌曲（异步任务）。

参数类型必填说明
playlist_urlstring必填歌单链接
max_downloadint可选最多下载数量，0=不限制（默认 0）
curl -X POST http://localhost:8866/api/playlist/download \
  -H "Content-Type: application/json" \
  -d '{"playlist_url": "https://music.163.com/#/playlist?id=7583298906", "max_download": 10}'

📝 搜索歌词

POST /api/lyrics/search

搜索歌词，支持多个歌词 API 源。

参数类型必填说明
track_namestring必填歌曲名
artist_namestring必填歌手名
allowed_lyric_apisstring[]可选歌词 API 列表，如 ["searchbylrclibapig", "searchbylrclibapis"]
curl -X POST http://localhost:8866/api/lyrics/search \
  -H "Content-Type: application/json" \
  -d '{"track_name": "晴天", "artist_name": "周杰伦"}'

📊 任务管理

GET /api/tasks

获取所有任务列表。

参数 (Query)类型必填说明
statusstring可选按状态过滤: pending / running / completed / failed
typestring可选按类型过滤: search / download / search_and_download / playlist_download
limitint可选返回数量限制（默认 50）

GET /api/tasks/{task_id}

查询指定任务的详细状态和结果。

DELETE /api/tasks/{task_id}

删除指定任务记录。

📁 文件管理

GET /api/files/list

列出工作目录下已下载的音频文件。

参数 (Query)类型必填说明
work_dirstring可选工作目录路径（默认 musicdl_outputs）
recursivebool可选是否递归遍历子目录（默认 false）

GET /api/files/download?path={file_path}

下载指定路径的音频文件。

参数 (Query)类型必填说明
pathstring必填文件绝对路径（从任务结果中获取）

💚 健康检查

GET /api/health

检查服务健康状态。

curl http://localhost:8866/api/health
